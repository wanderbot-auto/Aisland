Skills 页面布局候选方案

目标是让用户快速判断“有哪些 Skill、装在哪个作用域、当前是否生效”，同时避免在设置页展示 SKILL.md 内容摘要。下面 4 个方案都保留路径优先、卡片呈现、作用域状态可视化，并遵循现有 Cyber-Minimalist 视觉系统。

全局 / Aisland 托管适合用户主动安装，可卸载，应放在最显眼状态。
用户级来自用户目录，通常只读，用路径解释来源。
项目级当前工作目录或项目环境，强调临时上下文。
仓库级优先级最高，适合用覆盖状态提醒用户。
